2.  **內容分析與替換：** 逐一讀取並分析這些檔案的內容，將所有 `bes-web` 的實例替換為 `bes-app`。替換時需注意上下文，確保語法正確性。

4.  **清理：** 移除舊的 `bes-web` 相關 Docker 映像和容器（如果存在）。
